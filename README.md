## Sobre o Projeto

Site completo e responsivo de uma imobiliária de luxo, com foco em design sofisticado, animações suaves e experiência de usuário refinada.

### Funcionalidades

- Hero com slider automático e transições animadas
- Busca de imóveis com overlay fullscreen
- Portfólio de propriedades com filtros por categoria
- Seção de bairros nobres
- Depoimentos de clientes
- Formulário de contato
- Newsletter
- Vídeo institucional com player integrado
- Menu responsivo com mega-dropdown
- Preloader animado

## Tecnologias

- **Next.js 16** — Framework React com App Router
- **TypeScript** — Tipagem estática
- **SCSS/Sass** — Estilos com variáveis, mixins e nesting
- **Tailwind CSS 4** — Classes utilitárias
- **Framer Motion** — Animações e transições
- **Lucide React** — Ícones

## Como rodar

```bash
npm install
npm run dev
```

Acesse **http://localhost:3000**
